# SREIC-Validation-Pipeline
FTIA uses this model to make asset-renewal investment decisions on a network where speed restrictions cost an estimated €75.3M in passenger delay annually. The quality of that decision depends on how well the model is validated. Moving from n = 7 to n = 100 moves the CI from ±0.45 to ±0.11. That's the difference between a defensible claim/guess.
